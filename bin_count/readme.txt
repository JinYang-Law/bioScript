问题： 判断一个数值落在那个区间的个数
详细描述：
1. 设定 bin = 1x, 4x ,10x, 50x, 100x, 200x, 300x, 400x, 500x；
2. 设定一组位点的深度： depths = 431,401,23,354,452,123,432,234,656,.....
要求:
1. 统计深度在的 分别1X~4X区间， 4X~10x区间 ...... 的位点的数目
2. 统计深度在的 分别1X以上， 4X以上 ...... 的位点的数目
应用 ： 
1. samtool depth 生成vcf , 进行分bin的统计， 比如1x 4x ,10x, 50x, 100x, 200x, 300x, 400x, 500x ... 的点的个数
2. 也可以应用 CNV 分bin 统计深度

解决方案
思路： 
1. 通过二分查找，确定每个深度在bin 的的下标；
2. 通过下标确定深度在bin 区间（左右边界）
比如 depth = 431  在bin 的index=8   (400x, 500x] += 1

复杂度： O(log2n)
